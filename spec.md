# Specification

## Summary
**Goal:** Redesign key UI components in ConnectSphere to give the app a visually distinct identity, moving away from layouts and patterns that resemble Instagram.

**Planned changes:**
- **StoriesStrip**: Replace circular avatar bubbles with rectangular/pill-shaped story cards showing a square thumbnail, username label, and a colored border or glow indicator for unseen stories (no ring style)
- **PostCard**: Move content (image/text) to the top, place author info in a compact inline row below content, display like/comment counts as pill badges, and replace the overflow menu with an inline action icon row
- **ProfilePage**: Replace the 3-column post grid with a masonry or 2-column asymmetric grid, display follower/following stats as large typographic counters in a horizontal band, and make the edit profile button full-width with gradient styling
- **BottomNav**: Replace the full-width flat tab bar with a floating pill-shaped nav bar with drop shadow and backdrop blur; active tab uses a filled background highlight; Create Post button is an elevated square icon within the pill
- **HomePage feed**: Use glass/gradient card panels with left-accented colored borders per post type, contain images within rounded card boundaries, and add a visual section divider between the stories strip and the feed
- **LandingPage hero**: Redesign as an asymmetric split-panel layout with logo/tagline on one side and a gradient feature showcase on the other; replace the generic CTA button with a distinctly shaped gradient button

**User-visible outcome:** The app's main screens (landing, home feed, profile, stories, and navigation) look visually unique and no longer resemble Instagram's layout patterns, while preserving all existing functionality and the purple-blue gradient theme.
